# pythonanywhere-forever

Renews the 3 months free period of a pythonanywhere account automatically. Uses simple requests instead of a headless browser.

## Behavior

The app simulates the following flow:

https://user-images.githubusercontent.com/26277944/202918914-c7e5accc-f721-488c-bf4d-5b5a81bee945.mp4

## Usage

This uses GitHub actions; set `PA_USERNAME` and `PA_PASSWORD` secrets in the repo's settings to match your pythonanywhere credentials.
